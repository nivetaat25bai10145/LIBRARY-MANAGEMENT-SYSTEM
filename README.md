*Project Title*
Library Management System

*Overview*
A simple Python program to manage library books and users. It supports adding, issuing, returning, and searching books.

*Features*
Add and remove books

Search books by title or author

Issue and return books

Track users and borrowing history

*Technologies Used*
Python (via Jupyter Notebook)

No external databases (uses in-memory data structures or CSV files)


Installation & Run
Make sure Python 3 is installed.

Download the project files.

Run the program:

bash
python main.py
Testing
Follow on-screen prompts to add, issue, return, and search books.

Verify data persistence by restarting the app.

Use simple test data like new users and sample books
