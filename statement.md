
Problem Statement
Users need a reliable system to manage library operations such as adding/removing books, issuing books, returning books, and searching the catalog. Manual processes are time-consuming, prone to errors, and lack proper record-keeping. A computerized system is essential to automate inventory management, track book borrowings and returns, and maintain accurate records for better library management.

Scope of the Project
The system will enable:

Adding and removing books and users

Searching books by title or author

Issuing and returning books

Maintaining user and book records securely with validation

Showing the list of books present in library 

Target Users
Librarians/Administrators

Library members/Students

High-Level Features
Create and manage user and book records

Issue and return books with real-time updates

Search and view available books

Generate and view borrowing history

Ensure data validation and error handling for smooth operation
